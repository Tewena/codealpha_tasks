# Probot – Productivity FAQ Bot

A simple Python bot that answers common productivity questions using keyword matching.

## Requirements
- Python 3.11 and above
- nltk

## How to Run
1. Install nltk:
    ```python
    import nltk
    nltk.download('punkt')
    ```
2. Run the script:
    ```bash
    python probot.py
    ```
- Type your question and get a productivity answer.

## Files
- `probot.py` – main script
